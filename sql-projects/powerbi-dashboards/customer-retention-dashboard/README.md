  Customer Retention Dashboard (Power BI)

This dashboard visualizes customer activity, retention patterns, and churn behavior over time. It is designed to complement the SQL case study in this portfolio by transforming the same business logic into an interactive, visual format.

   Purpose
The goal of this dashboard is to demonstrate:
- Data modeling and transformation in Power BI
- KPI creation for churn, retention, and active customers
- Visual storytelling for business stakeholders
- Ability to translate SQL logic into a BI environment

   Key Metrics
- Monthly Active Customers
- Churned Customers
- Retention Rate
- Revenue Trends (if included)

   Visuals Included
- Line chart showing retention over time
- Bar chart for churn events
- KPI cards for monthly totals
- Optional slicers for customer segments or time periods

- `.pbix` file (Power BI dashboard)
